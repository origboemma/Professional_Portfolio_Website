# 1. Purpose

The Project Roadmap provides a high-level view of the planned development journey for the Professional Portfolio Website.

It identifies the major phases, milestones, deliverables, and long-term enhancements that will guide the project from planning through deployment and future evolution.

Unlike a detailed project schedule, the roadmap focuses on strategic direction rather than day-to-day activities.

---

# 2. Roadmap Objectives

The roadmap aims to:

- Maintain a clear long-term vision.
- Monitor progress across all development phases.
- Define major project milestones.
- Support planning and prioritisation.
- Identify future enhancements.
- Reduce project drift.

The next major activities include:

- Complete remaining UI/UX documents.
- Conduct the Design Review.

- Develop all website pages.
- Integrate project content.

# 7. Future Enhancements

Following Version 1.0, potential enhancements include:

- Technical blog and knowledge hub.
- Search functionality.
- Project filtering by technology and domain.
- Dark/Light mode toggle.
- Downloadable résumé section.
- Interactive project demonstrations.
- Case study comparison feature.
- Multilingual support.
- Analytics dashboard for visitor insights.
- AI-powered project assistant.

Future enhancements will be evaluated according to project priorities and user needs.
