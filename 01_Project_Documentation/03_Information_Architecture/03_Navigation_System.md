# 3. Navigation System

## 3.1 Purpose

This document defines the navigation system for the Professional Portfolio Website.

The navigation system enables visitors to move naturally throughout the website while maintaining orientation, minimising effort, and encouraging exploration of Emmanuel Chukwuka Origbo's professional work.

Navigation should be intuitive, consistent, responsive, and accessible across all devices.

---

# 3.2 Navigation Philosophy

The navigation system is guided by one primary principle:

> **Every click should move the visitor closer to understanding Emmanuel's professional value.**

Navigation should never distract from the content.

Instead, it should quietly support the visitor's journey from curiosity to confidence.

---

# 3.3 Primary Navigation

The primary navigation appears in the website header and remains consistent across every page.

The navigation menu consists of:

- Home
- About
- Professional Journey
- Projects
- Skills & Technologies
- Certifications
- Experience
- Resume
- Contact

The navigation should always remain visible on desktop devices.

---

# 3.4 Homepage Navigation

The homepage acts as an executive overview.

Navigation links direct visitors to dedicated pages rather than forcing all information onto one scrolling page.

Homepage preview sections include "Learn More" buttons that guide visitors naturally into the full content.

Examples include:

- About → Learn More
- Professional Journey → Read My Story
- Featured Projects → View All Projects
- Skills → Explore Skills
- Certifications → View Certifications
- Experience → View Experience
- Contact → Get In Touch

---

# 3.5 Project Navigation

The Projects page serves as the central directory for all portfolio projects.

Each project card links directly to a dedicated project case study.

Every project case study should include:

- Previous Project
- Next Project
- Back to Projects

This allows visitors to continue exploring without returning to the homepage.

---

# 3.6 Breadcrumb Navigation

Project pages should display breadcrumb navigation.

Example:

```text
Home > Projects > PrimeMart Retail Sales Operations Reporting System
```

Breadcrumbs help visitors understand where they are within the website structure.

---

# 3.7 Footer Navigation

The website footer should provide quick access to:

- About
- Projects
- Skills
- Certifications
- Experience
- Resume
- Contact

The footer should also include:

- GitHub
- LinkedIn
- Email
- Copyright Information

---

# 3.8 Mobile Navigation

On mobile devices, the navigation should collapse into a hamburger menu.

The mobile navigation should:

- Display all primary menu items.
- Be easy to open and close.
- Support touch interaction.
- Maintain the same page order as the desktop version.

Consistency between desktop and mobile navigation is essential.

---

# 3.9 Navigation Behaviour

Navigation should:

- Clearly indicate the active page.
- Highlight links on hover.
- Support keyboard navigation.
- Maintain smooth page transitions.
- Preserve consistent positioning throughout the website.

Visitors should always know their current location.

---

# 3.10 Call-to-Action Navigation

Strategic Call-to-Action (CTA) buttons should appear throughout the website.

Examples include:

- Explore My Projects
- Read My Journey
- Download Resume
- View GitHub
- Connect on LinkedIn
- Contact Me

These actions encourage meaningful engagement without interrupting the browsing experience.

---

# 3.11 Navigation Accessibility

The navigation system should:

- Support keyboard navigation.
- Include descriptive link labels.
- Maintain sufficient colour contrast.
- Provide visible focus indicators.
- Use touch-friendly button sizes.
- Avoid relying solely on colour to indicate active links.

Accessibility should be integrated into the navigation design from the beginning.

---

# 3.12 Navigation Summary

The navigation system provides a consistent and intuitive framework that guides visitors through the Professional Portfolio Website. By combining persistent navigation, contextual links, breadcrumbs, responsive menus, and strategic calls to action, the website supports an engaging user experience while reinforcing Emmanuel Chukwuka Origbo's professional brand.
