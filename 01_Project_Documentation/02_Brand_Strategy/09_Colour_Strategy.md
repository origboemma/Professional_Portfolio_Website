# 9. Colour Strategy

## 9.1 Purpose

This document defines the colour system for the Professional Portfolio Website. The selected colours are intended to strengthen the professional brand, improve usability, ensure visual consistency, and create an engaging experience for visitors.

The colour strategy prioritises professionalism, accessibility, readability, and long-term maintainability.

---

# 9.2 Colour Philosophy

The website will adopt a modern, technology-inspired colour palette that communicates:

- Professionalism
- Trust
- Intelligence
- Innovation
- Stability
- Simplicity

Colours should support the content rather than distract from it.

# 9.7 Colour Usage Guidelines

The website should maintain approximately the following balance:

- 70% Neutral Colours
- 20% Primary Colours
- 10% Accent Colours

This creates a clean, premium appearance while preventing visual overload.

---

# 9.8 Accessibility Considerations

The selected colour palette should:

- Maintain strong text contrast.
- Avoid relying on colour alone to communicate meaning.
- Support users with colour vision deficiencies.
- Follow recognised accessibility recommendations where practical.

Accessibility should always take priority over visual decoration.

---

# 9.9 Dark Mode Compatibility

The colour system should be designed so that a future dark mode can be implemented without redesigning the entire website.

Colours should therefore be organised using reusable design tokens or CSS custom properties during implementation.

---

# 9.10 Colour Summary

The Professional Portfolio Website adopts a balanced, technology-inspired colour system centred on blue, supported by neutral backgrounds and carefully selected accent colours. This strategy reinforces professionalism, enhances readability, improves accessibility, and establishes a strong visual identity across every page of the website.
