# 10. Typography Strategy

## 10.1 Purpose

This document defines the typography system for the Professional Portfolio Website. The typography strategy is designed to maximise readability, improve accessibility, establish a clear visual hierarchy, and reinforce Emmanuel Chukwuka Origbo's professional brand.

Typography should support communication rather than compete for attention.

---

# 10.2 Typography Philosophy

The website will use modern, clean, and highly readable typography that reflects professionalism, technical competence, and structured thinking.

The typography system should:

- Improve readability.
- Create visual consistency.
- Guide users naturally through content.
- Support accessibility.
- Present information in a clear hierarchy.

---

# 10.3 Primary Typeface

### Primary Font Family

**Inter**

Inter has been selected because it offers:

- Excellent screen readability.
- Modern appearance.
- Wide browser support.
- Professional aesthetic.
- Outstanding performance for digital interfaces.
- Comprehensive character support.

It will serve as the primary font for the entire website.

---

# 10.4 Fallback Font Stack

The website should use the following font stack:

```css
font-family:
"Inter",
system-ui,
-apple-system,
BlinkMacSystemFont,
"Segoe UI",
Roboto,
Arial,
sans-serif;
```

This ensures a consistent appearance across different operating systems and browsers.

---

# 10.5 Font Weight System

| Usage | Weight |
|---------|-------:|
| Large Headings | 700 |
| Section Headings | 600 |
| Card Titles | 600 |
| Navigation | 500 |
| Body Text | 400 |
| Captions | 400 |
| Buttons | 500 |

Using a limited range of font weights creates a clean and consistent design.

---

# 10.6 Typography Scale

| Element | Size |
|---------|------|
| Hero Title | 56px |
| Page Title | 40px |
| Section Heading | 32px |
| Subheading | 24px |
| Card Title | 20px |
| Body Text | 16px |
| Small Text | 14px |
| Caption | 12px |

This scale provides a clear visual hierarchy while remaining responsive.

---

# 10.7 Line Height

Recommended line heights:

| Content | Line Height |
|----------|------------|
| Headings | 1.2 |
| Body Text | 1.6 |
| Lists | 1.7 |
| Long Articles | 1.8 |

Proper spacing improves readability and reduces visual fatigue.

---

# 10.8 Typography Usage Guidelines

The typography system should follow these principles:

- Use sentence case for headings.
- Keep paragraph lengths manageable.
- Maintain consistent spacing.
- Avoid excessive bold text.
- Avoid unnecessary italic text.
- Use whitespace to separate ideas.
- Prioritise readability over decoration.

---

# 10.9 Responsive Typography

Typography should scale naturally across devices.

### Desktop

Use the full typography scale.

### Tablet

Reduce heading sizes proportionally while maintaining hierarchy.

### Mobile

Maintain readability with:

- Hero headings approximately 36–40px.
- Section headings approximately 28px.
- Body text remaining at 16px.

---

# 10.10 Accessibility Considerations

The typography system should:

- Maintain sufficient colour contrast.
- Avoid very small text.
- Support browser zoom.
- Preserve readability across screen sizes.
- Avoid decorative fonts for body content.

Typography should contribute to an inclusive user experience.

---

# 10.11 Typography Summary

The Professional Portfolio Website adopts a modern typography system centred on the Inter font family. Combined with a structured hierarchy, responsive sizing, and accessibility-focused design, the typography strategy supports clear communication, strengthens the professional brand, and provides a consistent reading experience across the website.
